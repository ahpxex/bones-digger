### 现场 Q&A 高频问题预案
1. "你们自己训模型了吗？"
   → "没训专门模型——RAG + 最新代 VLM 的组合证明：在专家知识库充分的领域，不需要从头训练就能达到可用水平。训练成本省下来给数据扩充。"
2. "跨物种相似样本怎么办？"
   → "精推通道。低置信度自动触发 397B Thinking，思维链可审计。"
3. "评委现场上传一张非骨骼图会怎样？"
   → OOD 兜底。现场 live 演示给评委看。
4. "和 GPT-4V 直接问有啥区别？"
   → 可解释性 + 知识硬约束 + 双通道 + 领域专用 UI（第 16 页对比表）
5. "300 张数据够吗？"
   → 我们不训模型，数据够 demo 验证。真实落地需要扩到 7 × 500 张（未来路线图第 1 条）
